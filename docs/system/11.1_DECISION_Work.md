# 11_DECISION

## Статус: ✅

## Описание

Этап 11_DECISION реализует минимальную функцию выбора паттерна реакции на основе activated_memory + meaning. Функция `decide_response` в `src/decision/decision.py` выбирает один из фиксированных вариантов: "ignore", "absorb", "dampen", "amplify".

### Принципы

- **Детерминированный выбор**: Без randomness или LLM, только на основе данных.
- **Интеграция с activated_memory**: Если max significance в activated > 0.5, выбирает "dampen" (опыт учит смягчать удар).
- **Fallback на Meaning**: Иначе использует логику Meaning (ignore/absorb по significance).
- **Автономность**: Вызывается после activation в runtime loop, модифицирует impact перед apply_delta.

### Пример

Для события с высокой activated significance (e.g., повторяющийся shock) выбирается "dampen", уменьшая impact на 50%.
