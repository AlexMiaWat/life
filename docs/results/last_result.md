## Ключевые обновления в документации

### Новые архитектурные решения (ADR)
- **ADR 009: Lifecycle Manager** - жизненный цикл системы (init/run/degrade)
- **ADR 010: Subjective Time** - расширенная реализация субъективного времени
- **ADR 011: SelfState Improvements** - валидация и защита состояния
- **ADR 012: Memory v2.0** - расширение памяти с забыванием и архивацией
- **ADR 013: Learning Layer** - слой обучения без оптимизации
- **ADR 014: Adaptation Layer** - слой адаптации поведения
- **ADR 015: Runtime Loop Refactoring** - рефакторинг runtime loop с менеджерами

### Новые компоненты
- **Comparison System v1.0** - система сравнения жизней с параллельным запуском инстансов
- **Пример использования Comparison System** - практическое руководство по сравнению жизней
- **Обновленная документация компонентов** - добавлены новые возможности и примеры

### Архитектурные улучшения
- **Рефакторинг Runtime Loop** - выделение менеджеров (SnapshotManager, LogManager, LifePolicy)
