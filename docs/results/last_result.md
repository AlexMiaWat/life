## Ключевые изменения в документации

### Архитектурные принципы
- **Истинная пассивность**: система только принимает данные при явных вызовах
- **Только raw counters**: исключены все derived metrics и интерпретации
- **Внешняя интеграция**: полное отсутствие влияния на runtime loop

### Новые компоненты
- `PassiveDataSink` - реактивный сборщик данных без фоновых потоков
- `RawDataAccess` - доступ к сырым данным для внешнего анализа
- Удалены: AsyncPassiveObserver, все REST API endpoints
