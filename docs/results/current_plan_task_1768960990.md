# План выполнения задачи: Добавить примеры использования для каждого модуля

## Цель задачи
Добавить практические примеры использования для всех модулей проекта Life, чтобы улучшить документацию и помочь разработчикам быстрее освоить работу с каждым компонентом.

- Проект имеет 16+ модулей в `src/` директории

## Структура модулей проекта

### Основные модули:
1. **runtime/** - Runtime цикл и менеджеры (loop.py, snapshot_manager.py, log_manager.py, life_policy.py, subjective_time.py)
2. **state/** - Управление состоянием (self_state.py)
3. **environment/** - События и генератор (event.py, event_queue.py, generator.py, generator_cli.py)
4. **meaning/** - Интерпретация значений (meaning.py, engine.py)
5. **activation/** - Активация памяти (activation.py)
6. **decision/** - Принятие решений (decision.py)
7. **action/** - Выполнение действий (action.py)
8. **intelligence/** - Интеллект (intelligence.py)
9. **planning/** - Планирование (planning.py)
10. **memory/** - Память (memory.py)
11. **learning/** - Обучение (learning.py)
12. **adaptation/** - Адаптация (adaptation.py)
13. **feedback/** - Обратная связь (feedback.py)
14. **monitor/** - Мониторинг (console.py)
15. **observability/** - Система наблюдаемости (structured_logger.py)
16. **dev/** - Dev utilities (process_restarter.py)
